== Description ==

It solves the problem of many templates that do not have open graph from Facebook. Now you can set content to be displayed in sharing on Facebook.

== Installation ==

Upload the Facebook Open Graph by OAlves plugin to your blog and activate it.
1, 2, 3: You're done!

== Frequently Asked Questions ==

What this plugin do?
Insert Open Graph meta tags to you header.
